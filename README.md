The CampCommander App requirews:
Splash screen which lasts for 3 seconds then enters the main page
The main to have buttons to display add items and display the total items packed
The view screen to have a display gear list details and implement a back to base button
Have parallel arrays to store items details, categories and quantities
Further it is required to initialise arrays with sample data
Error handling and loop work is also required
